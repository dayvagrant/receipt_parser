## Система оценки
Оценка производится по 3 категориям:
1. Название товара
2. Бренд товара
3. Категория товара

Для оценки использовалась самописная accuracy:
-   за каждый верное угадывание +1 балл
-   за частично верное +0.5
-   за неверное +0

Затем общая оценка переводится в проценты.
